Research on Improving Live Multimedia Streaming Over P2P Networks

Background

This research aims to enhance live multimedia streaming over peer-to-peer (P2P) networks using a novel structure based on Fibonacci Heaps. Traditional tree-based or mesh-based streaming methods often face high latency or inefficient bandwidth usage, especially under dynamic conditions like user churn or network fluctuations.

Purpose

Improve live multimedia streaming by creating a hybrid push-pull system that forms an overlay network using Fibonacci Heaps...

Problem

Legacy tree/mesh approaches suffer from high latency or underutilized bandwidth...

Proposed Solution

  • Fibonacci Heaps based overlay network reorganizes in response to latency/bandwidth changes.
  • Leaf nodes effectively utilized, reducing freeloaders.
  • Minimizes latency and ensures near-optimal bandwidth usage.

Key Advantages

  • Reduced latency with bounded delivery delay.
  • Dynamic adaptability under changing conditions.
  • Scales to 1 million nodes in simulations.

Implementation & Evaluation

  • Simulations show robust performance under churn.
  • Near-optimal bandwidth usage in large-scale tests.
  • Outperforms standard tree and mesh in delay/upload efficiency.
