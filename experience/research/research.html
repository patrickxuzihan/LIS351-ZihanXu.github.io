Research on Improving Live Multimedia Streaming Over P2P Networks

Background

This research aims to enhance live multimedia streaming over peer-to-peer (P2P) networks using a novel structure based on Fibonacci Heaps. Traditional tree-based or mesh-based streaming methods often face high latency or inefficient bandwidth usage, especially under dynamic conditions like user churn or network fluctuations.

Purpose

Improve live multimedia streaming by creating a hybrid push-pull system that forms an overlay network using Fibonacci Heaps. Nodes are grouped based on distance from the source and upload capacity, and the network restructures dynamically to minimize latency and maximize bandwidth utilization.

Problem

Legacy tree/mesh approaches suffer from high latency or underutilized bandwidth, especially with unpredictable churn (users leaving/arriving) and fluctuating network conditions.

Proposed Solution

  • Fibonacci Heaps based overlay network that reorganizes in response to latency/bandwidth changes.
  • Leaf nodes effectively utilized, reducing “freeloaders.”
  • Minimizes latency and ensures near-optimal bandwidth usage.

Key Advantages

  • Reduced latency with bounded delivery delay.
  • Dynamic adaptability under changing conditions.
  • Better bandwidth usage than tree or mesh approaches.
  • Scales well, handling up to 1 million nodes in simulations.

Implementation & Evaluation

  • Simulations demonstrate robust performance under churn.
  • Near-optimal bandwidth usage and low latency in large-scale tests.
  • Outperforms standard tree and mesh in delay and upload efficiency metrics.
